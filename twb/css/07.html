RWD

As páginas da Web podem ser visualizadas em muitos dispositivos diferentes: desktops, tablets e telefones. Sua página web deve ter uma boa aparência e ser fácil de usar, independentemente do dispositivo.

Como funciona?

O design responsivo (RWD) faz com que sua página da web tenha uma boa aparência em todos os dispositivos.

O RWD responsivo usa apenas HTML e CSS.

Web design responsivo não é um programa ou JavaScript.

Desmaximize e modifique o tamanho da janela do navegador para ver os efeitos de renderização.
